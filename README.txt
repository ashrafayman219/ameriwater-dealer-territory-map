HOW TO USE:
-----------
1. Open "index.html" in your web browser (Chrome or Firefox recommended)
2. Wait while the map loads ZIP boundaries from ArcGIS (10-30 seconds)
3. Click on any colored territory to see dealer details
4. Use the buttons on the left to switch between dealer layers
5. Hover over territories for information

HOW TO UPDATE TERRITORIES:
--------------------------
1. Edit your Excel file: __AMERIWATER DEALER ZIP CODE COVERAGE.xlsx
2. Save it as CSV format (File > Save As > CSV)
3. In the map, click the "Upload CSV" button (orange button in control panel)
4. Select your CSV file
5. Map automatically updates colors with new data - Done!

TROUBLESHOOTING:
----------------
Problem: Map takes time to load
Solution: Normal - loading from ArcGIS service (10-30 seconds)

Problem: Some territories don't appear
Solution: Those ZIP codes may not exist in ArcGIS database

Problem: CSV upload shows error
Solution: Make sure CSV format matches original (8 columns)

Problem: Colors don't update
Solution: Refresh the page and try again
